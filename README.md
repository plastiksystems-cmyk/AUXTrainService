## AUXTRAINSERVICE
### Information
The useful .luau scripts are placed inside of the scripts folder.
### ~~Quick Setup~~
Removed as of V2, use .rbxm file instead.
### ~~Installation~~
Use Railman V2.rbxm
### Requirements
- Train MUST BE a Model
- Train MUST HAVE "Train" tag.
- Train MUST BE inside of Workspace
- The train's VehicleSeat MUST BE set as first child
- Train's wheels MUST BE named "WheelR" or "WheelL" based on the side of the train they are positioned on. If the controls are in reverse, then invert the names.
- The wheels MUST USE HingeConstraints, not SurfaceHinges.
### Attributes
- MaxSpeed : number (radians)
- ReverseSpeed : number (radians)
- Acceleration : number (studs/second²)
- Headcode : string (GENERATED AUTOMATICALLY)
### Configuration
- Default_HornID : number | string
Enter ur SoundID as property of it.
- TeamsOnly : boolean
Determines if TrainSecurityService will be enabled.
- TeamsAllowed : string
Enter ur allowed teams separated by ";"
- HeadcodePrefix : string
All the headcodes generated by OperationService will have this prefix
Example: RAILMAN_12
- HornCooldown
Default HornCooldown for all trains.
- GravityEnabled
If true, your train will slow down using the gravitational acceleration of 0.98
## Default Attributes are applied if one or more of the train's needed attributes are missing.
